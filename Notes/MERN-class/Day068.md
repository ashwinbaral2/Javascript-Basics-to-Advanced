# 🔎 Autosuggestion Feature

An **autosuggestion system** was implemented for location search.

### How It Works

1. User begins typing an address
2. The API sends a request to fetch suggestions
3. A dropdown list displays possible locations
4. The user selects the correct location

### Benefits

* Faster input
* Reduced typing effort
* More accurate location selection

# 🔁 Multi-Step Form Implementation

The project includes a **multi-step interface** for the request process.

### State Management

A React state variable was used:

```javascript
const [step, setStep] = useState(1);
```

### Purpose

The `setStep` function allows the application to move between steps.

Example flow:

```
Step 1 → Enter Pickup Location
Step 2 → Select Hospital
Step 3 → Confirm Request
```

This creates a **structured and user-friendly workflow**.

# 🖱 Event Handlers for User Interaction

Event handlers were used to capture user interactions.

### Example

When a user clicks a suggestion:

* The selected location is captured
* The value is inserted into the input field
* The next step in the process can begin

Example concept:

```javascript
onClick={() => handleLocationSelect(location)}
```

### Where Event Handlers Were Used

* Pickup location selection
* Hospital search selection
* Suggestion list interaction
