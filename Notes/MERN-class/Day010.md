## ðŸŸ¡ 2ï¸âƒ£ COMBINING MAP() AND FILTER()

**ðŸ“Œ KEY CONCEPT:**

You can **chain methods** when transforming data.

**ðŸ”· EXAMPLE:**

Filter even numbers, then double them:

```javascript

const processed = nums
  .filter((n) => n % 2 === 0)   // filter even
  .map((n) => n * 2);           // double them

console.log(processed); // [4, 8, 12]
```

**ðŸ’¡ WHY THIS MATTERS:**

This pattern is **extremely common in MERN stack apps** when transforming API data:
- Filtering users by status
- Mapping items to display
- Converting data formats

## ðŸŸ¡ 3ï¸âƒ£ ACCESSING INDEX & ENTIRE ARRAY

**ðŸ“Œ CALLBACK SIGNATURE:**

All three methods (`forEach()`, `map()`, `filter()`) share the same signature:

```javascript
(item, index, array) => { }
```

**ðŸ”· EXAMPLE:**

```javascript
arr.forEach((item, index, array) => {
  console.log(`Item: ${item}, Index: ${index}`);
  // 'array' refers to the original array
});
```

**ðŸ“Œ WHEN TO USE IT:**

- Check previous/next elements
- Compare items with something in the array
- Debug transformations
- Access the full context of iteration

**ðŸ“Œ WHAT IS REDUCE()?**

A powerful method to **boil an array down to a single value**.

**ðŸ“Œ COMMON USE CASES:**

- Grouping data
- Counting frequency of values
- Converting arrays into objects

**ðŸ”· SIMPLE EXAMPLE:**

```javascript

```
