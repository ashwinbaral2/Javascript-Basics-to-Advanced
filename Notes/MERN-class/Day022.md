## âš›ï¸ Types of Components

### 1. Stateless Functional Components
- Function-based
- No internal state (traditionally)
- Lightweight and clean
- Most commonly used today

```js
function Navbar() {
  return <nav>Navbar</nav>
}
```

---

### 2. Stateful Class Components

* Class-based
* Can manage internal state
* Older React pattern (still exists, less common)

```js
class Sidebar extends React.Component {
  render() {
    return <aside>Sidebar</aside>
  }
}
```

> Modern React favors **functional components + hooks**.

## ğŸ“¦ Export vs Export Default

### `export`

* Named export
* Can export multiple items from a file
* Must be imported using the same name

```js
export const Navbar = () => {}
```

```js
import { Navbar } from "./Navbar"
```

---

### `export default`

* Default export
* Only one per file
* Can be imported with **any name**

```js
export default Navbar
```

```js
import Navbar from "./Navbar"
```

> Using `export`, the component can be used in **any other page** by importing it.
