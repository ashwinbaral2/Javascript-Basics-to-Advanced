# 🌐 Fetching Data from Backend

Two main approaches were discussed for fetching data from the backend to the UI.

---

## 1️⃣ Server Side Rendering (SSR)

**SSR** means the server prepares the data and sends the **fully rendered page** to the client.

### Advantages

* Better **initial page load performance**
* Improved **SEO**
* Better **security perception**

Because the data is rendered on the server, it is **not directly visible in the browser’s network tab** in the same way as client-side API calls.

---

## 2️⃣ Client Side Fetching

Another common approach is **client-side data fetching** using React hooks like `useEffect`.

Example:

```javascript id="fd2ux3"
useEffect(() => {
  fetch("/api/ambulance")
    .then(res => res.json())
    .then(data => setAmbulance(data));
}, []);
```

### Characteristics

* Data is fetched **after the page loads**
* API requests are **visible in the browser network tab**
* Commonly used in **React applications**

# 🌍 Distance Calculation (Haversine Formula)

To calculate the **distance between two points on Earth**, the **Haversine formula** is used.

This formula is widely used in:

* Navigation systems
* GPS applications
* Geographic Information Systems (GIS)
* Map services

### Example Use Case

Finding the **distance between:**

* User pickup location
* Nearby hospital

### Example Implementation in JavaScript

```javascript id="rvv7k1"
function haversineDistance(lat1, lon1, lat2, lon2) {
  const R = 6371; // Earth's radius in km

  const dLat = (lat2 - lat1) * Math.PI / 180;
  const dLon = (lon2 - lon1) * Math.PI / 180;

  const a =
    Math.sin(dLat/2) * Math.sin(dLat/2) +
    Math.cos(lat1 * Math.PI/180) *
    Math.cos(lat2 * Math.PI/180) *
    Math.sin(dLon/2) * Math.sin(dLon/2);

  const c = 2 * Math.atan2(Math.sqrt(a), Math.sqrt(1-a));

  return R * c;
}
```
