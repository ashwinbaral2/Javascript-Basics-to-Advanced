## 🔗 MCP (Model Context Protocol) — ⭐ Presentation Topic

### What is MCP?

**Model Context Protocol (MCP)** is a standard that allows AI models to securely and consistently interact with:

* Files
* APIs
* Databases
* Developer tools

> 🧩 *MCP is like a **universal connector (USB‑C)** for AI models*

### Why MCP Matters

* No custom integration per model
* Same tools usable by multiple AI systems
* Clean separation of:

  * AI logic
  * Data access
  * Tool execution

### MCP vs Traditional Integrations

| Feature             | MCP | Custom APIs |
| ------------------- | --- | ----------- |
| Standardized        | ✅   | ❌           |
| Tool Discovery      | ✅   | ❌           |
| Cross‑model support | ✅   | ❌           |
| Secure permissions  | ✅   | ⚠️          |

## 🧠 RAG System (Quick Overview)

**RAG = Retrieval + Generation**

Steps:

1. User asks a question
2. Relevant data is retrieved (DB / vector store)
3. LLM generates an answer using that context

> Used heavily in **enterprise AI apps**
