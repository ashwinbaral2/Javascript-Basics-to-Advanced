I am building a medical emergency + hospital social web app called "Medlink".

---
### 1) Logo Section
- Top-left logo placeholder (Medlink).
- Clicking it routes to `/`.
- Logo should stay visible on desktop and mobile.

### 2) Primary Links (Desktop)
- Home ‚Üí `/`
- Dashboard ‚Üí `/dashboard`
- Map ‚Üí `/map`

  - Ambulance Request

  `/emergencies/blood`, `/emergencies/ambulance`, etc.

- The navbar should be designed so it works nicely on map pages:

  - Street
  - Satellite
  - Terrain

    - Logout

### 6) Notifications (Future-ready)
- Add a bell icon placeholder with a badge count.
- This will later show emergency alerts near the user.

### 7) Responsiveness
- Must collapse into hamburger menu on mobile.
- Mobile menu must include:
  - Home, Dashboard, Map, Fundraiser
  - Emergency categories list
  - Login/Register or Profile dropdown

### 8) Styling
- TailwindCSS only.
- Modern medical look:
  - clean spacing,
  - rounded buttons,
  - subtle shadow,
  - clear hover states.
- Use a sticky navbar (fixed at top) so it stays visible while scrolling.

### 9) Implementation Requirements
- Use functional React component.
