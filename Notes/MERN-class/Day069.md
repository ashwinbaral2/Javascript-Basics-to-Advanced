# 🧠 Project Discussion

## 1️⃣ `useEffect()` in React

The **`useEffect()` hook** in React is commonly used for:

**A. Data Fetching**

* Fetch API data when a component loads.

Example use cases:

* Calling APIs
* Loading user data
* Fetching map routes

**B. Animations / Re-renders**

* Trigger animation after component mounts
* Update UI when state changes

Typical pattern:

```
useEffect(() => {
   // fetch data or trigger animation
}, []);
```

The empty dependency array `[]` means:

➡️ Run **only once when component mounts**

## 3️⃣ Polyline in Maps

**Polyline** is used to draw **routes on maps**.

Example use case:

* Show the route between **pickup location → destination**

In **React Leaflet**, a polyline draws the path between coordinates.

Example concept:

```
<Polyline positions={routeCoordinates} />
```

Where `routeCoordinates` might look like:

```
[
  [27.7172, 85.3240],
  [27.7000, 85.3333]
]
```

Result:

🗺️ A **line on the map showing the travel route**.

# 🧩 Typical Flow in the Project

1️⃣ User selects **pickup location**
2️⃣ User selects **destination**
3️⃣ App sends coordinates to **Geoapify Route Matrix API**
4️⃣ API returns route coordinates
5️⃣ Coordinates are passed to **React Leaflet Polyline**
