# 1. Tracking Live User Location (Browser Geolocation API)

JavaScript provides a built-in **Geolocation API** to access a user's location through the browser.

### Get Current Location

```javascript
navigator.geolocation.getCurrentPosition((position) => {
  const latitude = position.coords.latitude;
  const longitude = position.coords.longitude;

  console.log(latitude, longitude);
});
````

### Key Points

* Requires **user permission**
* Works best on **HTTPS**
* Returns **latitude and longitude**

---

# 2. Continuous Location Tracking

To track the user's movement in real time (like ride apps or delivery apps):

```javascript
navigator.geolocation.watchPosition((position) => {
  const lat = position.coords.latitude;
  const lng = position.coords.longitude;

  console.log(lat, lng);
});
```

| Method                 | Purpose                     |
| ---------------------- | --------------------------- |
| `getCurrentPosition()` | Fetch location once         |
| `watchPosition()`      | Track location continuously |

# 3. Reverse Geocoding (Coordinates → Address)

# 8. React Hooks Used

## useMemo()

Used to **memoize expensive calculations**.

```javascript
const memoizedValue = useMemo(() => {
  return heavyCalculation(data);
}, [data]);
```

### Benefits

* Prevents unnecessary recalculation
* Improves performance

---

## useRef()

Used to store a **persistent mutable reference**.

```javascript
const mapRef = useRef(null);
```

### Common Uses

* Access DOM elements
* Store map instance
* Store timers or intervals
* Prevent unnecessary re-renders
