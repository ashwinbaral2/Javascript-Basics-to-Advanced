## UI Component Libraries & Design Systems

### shadcn/ui
- Using **shadcn/ui** for frontend projects.
- Emphasis on **accessible, composable, and unopinionated components**.
- Encourages understanding of **component structure**, not just copy-paste usage.
- Recommendation:  
  - Explore and understand the **directory structure** (`components`, `lib`, `hooks`) to maintain scalability and consistency across projects.

### Starslab UI
- A more **niche UI library** focused on unique and less conventional design patterns.
- Useful for standing out visually when default UI libraries feel repetitive.
- Good candidate for experimental or portfolio projects.

### 8bitcn (Gaming UI)
- Website: https://www.8bitcn.com/
- Focuses on **retro / gaming-inspired UI components**.
- Helpful for projects involving:
  - Games
  - Gamified dashboards
  - Nostalgic or pixel-art aesthetics
- Reinforces the idea that UI should match the *theme and audience*, not just look ‚Äúmodern‚Äù.

## AI & Developer Tools (Trainer Recommendations)

### AI SDK
- https://ai-sdk.dev/
- Provides tools for **integrating AI capabilities** directly into applications.
- Useful for building AI-assisted features without reinventing infrastructure.

### Algolia Site Search
- https://sitesearch.algolia.com/
- Industry-grade **search-as-a-service**.
- Ideal for:
  - Documentation sites
  - Content-heavy platforms
  - Improving UX with fast and relevant search results

## AI-Assisted UI Generation

### v0 by Vercel
- https://v0.app/
- AI-powered tool for generating UI components from prompts.
- Useful for:
  - Rapid prototyping
  - Exploring layout ideas
- Important note: Treat as a **starting point**, not a replacement for design thinking or clean code.

## UI Inspiration & Design Review

### Dribbble
- https://dribbble.com/
- Platform to explore **UI/UX concepts across different project types**.
- Best used for:
  - Design inspiration
  - Understanding visual trends
- Caution: Designs are often **conceptual**, so always evaluate feasibility and accessibility before implementation.
