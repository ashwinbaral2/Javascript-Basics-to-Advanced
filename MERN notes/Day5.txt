
        *Git Features (Quick Revision Notes)*
        =====================================

1) Project Setup & Basics
-------------------------
git init                      → start new repo
git clone repo-link           → copy remote repo locally
git remote -v                 → show remote connections

2) ACP Rule (Add → Commit → Push)
---------------------------------
git add .                     → stage all changes
git commit -m "msg"           → save staged changes
git push origin yourbranch    → upload commits
git status                    → show changes/staging info
git restore filename          → restore file to last commit
git reset                     → undo commits/unstage changes (local)

3) Branching & Navigation
-------------------------
git branch                    → list/create/delete branches
git checkout branch           → switch branches
git checkout -b newbranch     → create + switch
git mv                        → move/rename files
git rm filename               → delete file and stage

4) Getting Updates
------------------
git pull                      → fetch + merge updates
git fetch                     → fetch only (no merge)
git pull origin branch        → pull from specific branch

5) Merging & Integration
------------------------
git merge yourbranch          → merge branch into current
git rebase yourbranch         → cleaner merge (rewrite commits)
git cherry-pick commitID      → copy a specific commit into your branch

6) History, Inspection, Debugging
---------------------------------
git log                       → view commit history
git log --oneline             → compact history
git diff                      → compare changes
git show commitID             → details of specific commit
git blame filename            → see who edited each line
git tag v1.0.0                → mark version/release

7) Undoing & Cleanup
--------------------
git revert commitID           → undo safely with new commit
git reset                     → undo commits locally
git restore filename          → restore file
git stash                     → temporarily save uncommitted work
git clean                     → remove untracked files


===============================
 Terminal Commands — Quick Groups
===============================

1) Moving Around
----------------
cd folder                     → go inside folder
cd ..                         → go back
pwd                           → show current path

2) Viewing Files
----------------
ls                            → list files/folders
ls -force                     → list including hidden

3) Creating & Editing
---------------------
mkdir foldername              → make new folder
touch filename                → create new file
code .                        → open folder in VS Code

4) Copy / Move / Delete
------------------------
cp source dest                → copy files/folders
mv old new                    → rename/move
rm filename                   → delete file
rm -r foldername              → delete folder (recursive)

5) Running Code
---------------
node filename.js              → run JS file
npm install package           → install npm package
npm start                     → start project (if script exists)
