## ðŸŽ¨ CSS: `z-index`

- Controls the vertical stacking order of elements.
- Works only on positioned elements:
  - `position: relative`
  - `position: absolute`
  - `position: fixed`
- Higher `z-index` values appear on top.

**Important in Map-Based Applications For**
- Popups
- Modals
- Toolbars layered above maps
