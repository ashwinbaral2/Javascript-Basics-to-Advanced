## useState() Hook

* A **React Hook** used to manage state inside functional components.
* State = data that **changes over time** and affects UI rendering.
* When state updates, **React re-renders the UI** automatically.
* Helps keep UI and data in sync without manually touching the DOM.

> Think of hooks as **batteries**: they power heavy lifting like UI updates so you donâ€™t have to.

## Hooks (Big Picture)

* Hooks let functional components do things that were once class-only.
* Examples: state, side effects, context, refs.
* They **simplify logic reuse** and keep components cleaner.
* Heavy tasks like UI updates and lifecycle handling are abstracted away.

## Importing & Using Components as Tags

* React components can be used like **custom HTML tags**.
* Must:

  * Export the component
  * Import it where needed
* Component names must start with **capital letters**.

This enables:

* Reusable UI blocks
* Clean and readable JSX
